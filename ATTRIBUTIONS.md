# Attributions

## Original project

This system is a substantially reworked fork of the original
**OpenD6 Space** Foundry VTT system by **Jim Raney** (GitLab:
[vtt2](https://gitlab.com/vtt2)), released under the MIT License.
The original project is at
<https://gitlab.com/vtt2/opend6-space>.

The codebase has since been rewritten — TypeScript, esbuild,
ApplicationV2, TypeDataModel, Scene Regions — but the original
work provided the foundation and the compendium content that this
system builds on.
